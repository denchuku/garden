## Pass argument to stdin

### Option 1

````
echo "hello" | base64
````

### Option 2

````
base64 <<< "hello"
````

### Option 3 (manual)

````
base64
````

Then enter `hello`, `↵` to go on new (empty line), press `⌃` + `d` to EOF standard input
